## Orders App

This is an app built using the MEAN stack. It implements a RESTful API to communicate to an AngularJS front end. It allows you to view, add, delete and search orders.

http://orders.zidaroiu.com

### Requirements

- node
- MongoDB
- Mocha
- Grunt

### Installation

``` nvm use ```
``` npm i ```

### Start server

``` npm start ```

### Build

``` npm run build ```

### Test

``` npm test ```
